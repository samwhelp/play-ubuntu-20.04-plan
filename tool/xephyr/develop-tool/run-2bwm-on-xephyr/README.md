
# run-2bwm-on-xephyr


## Howto

Put script to [~/bin/].


## Help

run

``` sh
$ make
```

run

``` sh
$ make help
```

show

```
Usage:
	$ make [action]

Example:
	$ make
	$ make help

	$ make install

```


## Install

run

``` sh
$ make install
```
