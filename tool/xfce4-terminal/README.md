
# Tool / xfce4-terminal



## Config Dir

* [~/.config/xfce4/terminal](config/xfce4-terminal)



## Config File

* [~/.config/xfce4/terminal/terminalrc](config/xfce4-terminal/terminalrc)


## Usage

### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```
