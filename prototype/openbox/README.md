## Usage


### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```


## Keybind and Mousebind

Please refer to

* $ less [~/.config/openbox/rc.xml](config/openbox/rc.xml)
